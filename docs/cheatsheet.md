チートシート
=========================

フロントエンド
-------------------------

### 開発環境の起動

```sh
npm run dev
```

### テスト

```sh
npm run test
```

### ビルド

```sh
npm run build
```

バックエンド
-------------------------

### 作成/起動

```sh
vagrant up
```

### 停止

```sh
vagrant halt
```

### 構築

```sh
vagrant provision
```

### 削除

```sh
vagrant destroy
```
